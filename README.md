# 暖心便签弹窗阵列 🎉

一个充满温暖与关怀的Python桌面弹窗程序，用精美的圆角弹窗展示暖心话语，适合作为生日祝福或日常温馨提醒。

## ✨ 特性

- 🎨 **精美圆角设计**：所有弹窗均采用大圆角设计，美观大方
- 🌈 **柔和配色**：随机生成柔和的马卡龙色系背景
- 📝 **双语支持**：包含中英文暖心话语
  - 50条中文关怀话语（包含13条毛泽东语录）
  - 50条英文温馨话语
- 🎯 **智能字体**：自动识别语言，中文使用宋体，英文使用Times New Roman
- 🎊 **生日祝福模式**：最后显示居中的生日祝福弹窗，带统一关闭按钮
- ⚙️ **灵活配置**：支持自定义弹窗数量和显示间隔

## 📸 预览

程序运行时会：
1. 随机在屏幕上弹出150个精美的圆角便签窗口
2. 每个窗口显示一条独特的关怀话语
3. 最后在屏幕中央显示生日祝福，可一键关闭所有窗口

## 🚀 使用方法

### 方式一：直接运行打包好的exe文件

下载 `dist` 文件夹中的 `暖心便签弹窗阵列.exe`，双击运行即可！

**命令行参数：**

```bash
# 快速测试（5个弹窗）
暖心便签弹窗阵列.exe --test

# 默认运行（150个弹窗）
暖心便签弹窗阵列.exe

# 自定义数量（例如100个）
暖心便签弹窗阵列.exe --count 100

# 调整显示速度（毫秒间隔）
暖心便签弹窗阵列.exe --count 150 --stagger 50
```

### 方式二：从源码运行

**环境要求：**
- Python 3.8+
- tkinter（Python自带，无需额外安装）

**运行命令：**

```bash
# 测试运行
python 暖心便签弹窗阵列.py --test

# 完整运行
python 暖心便签弹窗阵列.py

# 自定义配置
python 暖心便签弹窗阵列.py --count 200 --stagger 80
```

## 🛠️ 自己打包

如果你想自己打包程序，可以使用以下方法：

### 安装PyInstaller

```bash
pip install pyinstaller
```

### 打包为单文件exe

```bash
# 方法一：使用提供的spec文件（推荐）
pyinstaller 暖心便签弹窗阵列.spec

# 方法二：使用PowerShell脚本（Windows）
.\build.ps1
```

打包完成后，可执行文件位于 `dist` 目录中。

## 📋 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--count N` | 弹窗数量 | 150 |
| `--stagger ms` | 每个弹窗间隔（毫秒） | 80 |
| `--test` | 测试模式（只显示5个弹窗） | - |

## 🎨 技术特点

- **tkinter GUI**：使用Python标准库创建窗口界面
- **透明窗口技术**：通过透明色实现无边框圆角效果
- **Canvas绘图**：使用Canvas绘制圆角矩形
- **随机布局**：弹窗随机分布在屏幕上，不重叠主要区域
- **统一关闭机制**：最后的生日弹窗可关闭所有窗口

## 📝 消息内容

程序内置100条精心挑选的暖心话语：

- **中文消息（50条）**
  - 13条毛泽东经典语录
  - 37条日常关怀体贴话语
  
- **英文消息（50条）**
  - 50条温馨关怀短语

所有消息都经过精心挑选，传递正能量和关怀。

## ⚠️ 注意事项

1. 运行时会创建大量弹窗，建议先使用 `--test` 参数测试
2. 弹窗不会自动关闭，需点击最后出现的生日弹窗上的 `X` 按钮统一关闭
3. 如果弹窗数量过多可能影响电脑性能，建议根据配置调整数量
4. 单文件exe体积约10MB，包含了完整的Python运行时和tkinter库
